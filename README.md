Latin Parser & Flashcards App

An interactive, single-page web application designed to help students learn Latin grammar, morphology, and translation through a smart flashcard and quiz system.

🌟 Features

Interactive Flashcards: Learn Latin sentences word by word. Each word is broken down into its morphological properties (Noun/Verb/Adjective, Declension, Gender, Number, Case, etc.).

Detailed Morphological Explanations: Click on any grammatical tag to read a detailed, logical explanation of why the word takes that specific form or ending.

Smart Quiz Mode (Review): Test your knowledge! The app quizzes you on sentence translations and the exact morphological parsing of individual words with randomized, context-aware multiple-choice questions.

Advanced Text-to-Speech (TTS): * Automatically reads Latin and Arabic text.

Smart voice detection: Automatically selects the best available voices based on your browser (e.g., Google voices on Chrome, Microsoft/Isabella/Shaker voices on Edge).

Independent speed controls for Latin and Arabic audio.

Auto-Play Engine: A built-in loop and auto-advance system that reads the flashcards continuously, hands-free.

Export to PDF: Save all flashcards as a clean, landscape-formatted PDF for offline studying and printing.

Dark/Light Mode: Fully responsive UI with a premium dark and light theme toggle.

🚀 How to Use

This project is entirely self-contained in a single HTML file. No build tools, package managers, or server setups are required!

Download or clone this repository.

Open the index.html file in any modern web browser.

Note: For the absolute best Text-to-Speech (TTS) experience, we highly recommend using Microsoft Edge or Google Chrome.

Toggle between "شرح" (Study Mode) to read and listen to the flashcards, and "مراجعة" (Quiz Mode) to test your skills.

🛠️ Tech Stack

React 18 (Loaded via CDN)

Tailwind CSS (For rapid, responsive UI styling)

Babel Standalone (For in-browser JSX compilation)

Web Speech API (For native Text-to-Speech capabilities)

🖨️ Printing to PDF

To save the flashcards as a PDF:

Go to the "شرح" (Study Mode) tab.

Click the Printer Icon at the top right.

The app will automatically format the cards into a clean, distraction-free landscape layout. Select "Save as PDF" in your print dialog.
